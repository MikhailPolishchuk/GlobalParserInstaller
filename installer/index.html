Установка GlobalParserInstaller

Порядок установки

  1. Скачайте сертификат.
  2. Установите сертификат в Локальный компьютер → Доверенные корневые центры сертификации.
  3. Установите тот же сертификат ещё раз в Локальный компьютер → Доверенные издатели.
  4. После этого запустите установку приложения.

Как установить сертификат

  1. Откройте файл GlobalParserInstaller.cer.
  2. Нажмите Установить сертификат / Install Certificate.
  3. Выберите Локальный компьютер / Local Machine.
  4. Выберите Поместить все сертификаты в следующее хранилище.
  5. Нажмите Обзор.
  6. Выберите Доверенные корневые центры сертификации.
  7. Завершите установку.
  8. Повторите те же шаги ещё раз, но на шаге выбора хранилища выберите Доверенные издатели.

Если установка приложения всё ещё недоступна

  • Закройте окно установки приложения.
  • Повторно откройте файл GlobalParserInstaller.appinstaller.
  • Убедитесь, что сертификат установлен именно в Локальный компьютер, а не в Текущий пользователь.
Важно: если сертификат не установлен в нужные хранилища, Windows может показывать издателя как Unknown, а кнопка установки приложения будет неактивной.
