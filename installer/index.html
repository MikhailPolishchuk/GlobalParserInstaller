Установка GlobalParserInstaller

Порядок установки

  1. Скачайте сертификат.
  2. Откройте файл сертификата и установите его в доверенные хранилища Windows.
  3. После этого запустите установку приложения.
Скачать сертификат Установить приложение

Как установить сертификат

  1. Откройте файл GlobalParserInstaller.cer.
  2. Нажмите Install Certificate.
  3. Выберите Local Machine.
  4. Выберите Place all certificates in the following store.
  5. Установите сертификат в Trusted Root Certification Authorities.
  6. При необходимости также установите его в Trusted Publishers.
Важно: если сертификат не установлен, Windows может показать издателя как Unknown, а кнопка установки приложения может быть недоступна.

Версия установщика: 1.0.8.0
